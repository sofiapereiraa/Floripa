# 🌴 Floripa - Site sobre Florianópolis

Este é um projeto de site desenvolvido com HTML e CSS sobre a cidade de Florianópolis, conhecida como a *Ilha da Magia*.

## 📌 Sobre o projeto

O site apresenta informações sobre Florianópolis, incluindo sua história, cultura e pontos turísticos mais famosos.  
O objetivo é praticar desenvolvimento web e organização de páginas usando HTML e CSS.

## 🌐 Acesse o site

👉 https://sofiapereiraa.github.io/Floripa/

## 🧱 Tecnologias utilizadas

- HTML5  
- CSS3  

## 📸 Conteúdo do site

- Página inicial com boas-vindas  
- Seção sobre a cidade  
- Informações turísticas  
- Imagens representando Florianópolis  

## 🎯 Objetivo

Este projeto foi criado para treinar:
- Estrutura de páginas HTML  
- Estilização com CSS  
- Organização de arquivos  
- Publicação no GitHub Pages
